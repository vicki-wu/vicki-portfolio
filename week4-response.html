Week 4 Assignment

This website is built using HTML, CSS, and JavaScript. The HTML provides the structural framework, including the title, description, and links to various projects in Taper #11, such as "Hollywood Hitmaker," while CSS enhances the visual presentation with a full-width body, a dark header with light text, and distinct styles for text boxes and buttons. Between the script tags, the developer uses functions like randomizeTitles(), which selects random movie titles and descriptions from predefined lists to create a slot machine effect, and spinButtonClicked(), which handles the click event for the "SPIN" button, triggering the randomization process and displaying results. If the combination of "Oceans 11" appears, a conditional statement in spinButtonClicked() reveals a hidden message and resets the "Materialized Dreams" counter. Additionally, a blinking "HOLLYWOOD HITMAKER" title is controlled by a function using setInterval() to alternate its visibility every second, adding continuous movement to the page.
